    2019-10-17 04:03
    npm, 命令行

    npm常用命令 # npm更新 # npm install npm@latest -g # 最新稳定版全局安装 npm install npm@4.2.1 -g # 指定版本全局安装 包安装 # 首先初始化，在需要安装的目录下输入 npm init -y # 默认配置 安装 npm install xxx # 安装 npm i xxx # 简写 # 安装一些常用库: npm i jquery npm i vue npm i bootstrap # 安装指定版本 npm i jquery@3.1.0 在根目录下的 package.json 中有每个库的版本号，如 "dependencies": { "bootstrap": "^4.3.1", "jquery": "^3.4.1", "vue": "^2.6.10" } 当移动当前根目录的项目时，即时删除掉 node_modules 文件夹，只要在当前目录下执行： npm i # 注意命令后不要加任何其他的东西 即可恢复之前所有的库，版本也是之前对应的。 ...
