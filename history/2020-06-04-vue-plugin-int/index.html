    Sass 安装 #

    安装sass的依赖包

    npm install --save-dev sass-loader
# sass-loader 依赖于 node-sass
npm install --save-dev node-sass

    vue 中使用

    new webpack.ProvidePlugin({
  Vue: ['vue/dist/vue.esm.js', 'default']
});
